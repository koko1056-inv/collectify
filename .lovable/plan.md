

## アバタータブのUX改善プラン

### 現状の問題分析

現在の `アバター` タブには5つの主要な要素が混在しています：

1. **HeroAvatar** - 大きな現在のアバター + アクション（生成/着せ替え/ランダム/アップロード）
2. **ギャラリー** - 過去のアバター履歴
3. **AvatarSocialSection**（`AvatarSocialSection.tsx`）に含まれる：
   - いいね / シェアボタン（ローカル state のみで永続化されない仮実装）
   - 同じ推しのコレクター
   - 人気アバター
4. **HeroAvatar 内のランダムボタン**（`Dices` アイコン）

**問題点**：
- 「自分の分身を育てる」という主目的に対し、ソーシャル要素（いいね/シェア/他人のアバター）が混ざり、目的が散漫
- いいね・シェアが**仮実装**で永続化されておらず、UXの破綻リスク
- ランダムは「何が起こるか分からない」ボタンで初心者が迷う
- 1画面に CTA が多すぎて「次に何をすればいいか」が不明瞭

---

### 改善の方向性

**コンセプト：「育てる場」に特化する**

アバタータブは「自分の分身をつくる・育てる・着せ替える」という個人体験に集中させ、ソーシャル要素は別の場所（探索ページや他人のプロフィール）に委ねます。

### 具体的な変更

#### 1. 削除する要素

| 要素 | 場所 | 理由 |
|------|------|------|
| いいねボタン | `AvatarSocialSection` | 仮実装で機能していない・自分のアバターに自分でいいねする違和感 |
| シェアボタン | `AvatarSocialSection` | HeroAvatar 詳細やスタジオから個別にシェア可能。トップに常時置く必要なし |
| 同じ推しのコレクター | `AvatarSocialSection` | アバター文脈に無関係。「探す」タブの方が適切 |
| 人気アバター | `AvatarSocialSection` | 探索系コンテンツ。マイルームに置くと自分中心の体験を阻害 |
| ランダムボタン | `HeroAvatar` のアクション群 | 何が起きるか分かりにくく初心者が混乱 |

→ **`AvatarSocialSection.tsx` 自体を削除**し、`AvatarCenterHome.tsx` から import と参照も削除  
→ **`RandomPickupModal` の参照を削除**（ファイル自体は他で使う可能性があるため残す）  
→ HeroAvatar のアクションを **4つ → 3つ** に減らす

#### 2. アクション3つに整理（HeroAvatar下部）

```text
┌──────────────┬──────────────┬──────────────┐
│   ✨ 生成    │  👕 着せ替え │  ⬆ アップ   │
│  (AIで作る)  │  (主役・強調) │  (写真から)  │
└──────────────┴──────────────┴──────────────┘
```

- **着せ替え** を中央＆強調色（accent）にして、「グッズを使った最大の魅力」を前面に
- 各ボタンの下に1行のヒントテキストを追加（例：「AIで一から作る」「グッズで装う」「写真をそのまま使う」）

#### 3. 画面下部に「次にやること」のヒントカード（任意）

ソーシャルセクションを削除した分、空白になる下部に**シンプルな1枚のヒントカード**を追加：

- アバター未設定 → 「最初のアバターを作ってみよう」CTA
- アバター1枚のみ → 「グッズで着せ替えてみよう」CTA（→ 着せ替えタブを開く）
- アバター複数枚 → 何も表示しない（過剰な誘導を避ける）

これにより、初心者には進む道を示し、上級者には邪魔をしない設計になります。

### UX改善後の画面構成

```text
┌─────────────────────────────────┐
│ ✨ AI推しアバター     [+ 新規]  │  ← イントロ
├─────────────────────────────────┤
│                                 │
│      ┌──────────┐               │
│      │  アバター │               │  ← HeroAvatar
│      │  (大)    │               │
│      └──────────┘               │
│                                 │
│ [生成] [着せ替え★] [アップ]    │  ← 3アクション
├─────────────────────────────────┤
│ ギャラリー (3)         すべて → │  ← 履歴
│ [□] [□] [□]                  │
├─────────────────────────────────┤
│ 💡 グッズで着せ替えてみよう    │  ← ヒント(条件付き)
│ お気に入りグッズで装って ▶︎    │
└─────────────────────────────────┘
```

---

### 期待される効果

- **目的の明確化**：「自分のアバターを育てる」という1つの目的だけに集中
- **判断疲れの軽減**：アクション数が減り、次の一手が分かりやすい
- **見た目の整理**：縦に長く重い画面が、すっきりとした体験に
- **仮実装の削除**：機能していないいいね・シェアによる「壊れた印象」を排除
